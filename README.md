# 3dxtrsim
O ASD TUG SIM v3.3 é um simulador de manobras náuticas focado em rebocadores do tipo Azimuth Stern Drive (ASD). Desenvolvido em ambiente web sem dependência de motores gráficos de grande porte, a aplicação demonstra um excelente uso de tecnologias nativas do navegador para alcançar alta fidelidade física e visual.
